Development Production Line - The Short Story [pdf, single html and chunked html].

Just a setup manual triggered by a hard drive crash in 2007, which made it painfully obvious how much time it costs to install and get the different components up and running. It already proved to be helpful when upgrading those components.

Thanks to the unmentioned posters and bloggers whose comments and installation pointers helped getting everything up and running, but keeping track where I found all those little snippets would be a project in itself.

The JModalWindow project was created to support modal functionality, similar to JDialog, without blocking all frames.
